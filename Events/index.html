Triggering CSS Animations with JavaScript

Transitions

JavaScript

function addBorder() {
document.getElementById("test1").classList.add("addBorder");
}

CSS

.addBorder {
border-style: dashed; }

JavaScript

function changeColor() {
document.getElementById("colorEx").classList.add("changeColor");
}

CSS

.changeColor {
animation-name: colorEx;
animation-duration: 8s;
animation-iteration-count: infinite;}

JavaScript

function addFun() {
document.getElementById("funEx").classList.add("addFun");
}

CSS

.addFun {
