Loops

Loops are pieces of code that repeat a number of times. JavaScript supports several differnt loops.

1. For loop. This loops allows you to travel through the block of code a specific number of times.

var i;
for ( i = 1;  i < 6;   i++)    {

     statement  =   "Printed"  +  i  +  "times";

}


2. For/In loop. This loop allows you to travel through each propertie in a given object.

var grades = {Test1: "98", Quiz1: "68", Test2: "89"}
var text = "";
var x;
for ( x in grades)    {
     text += grades[x];
}


3. While loop. This loops allows you to travel through the block of code while a specific condition is met.

while (i < 5 )    {
    text += "The current number is " + i;
i ++;
}


4. Do/While loop. This loops allows you to travel through the block of code at least once and then while a specific condition is met.

do {
    text += "The current number is " + i;
i ++;
}
while (i < 5 );   

Conditional Statements

Conditional statements are pieces of code that compute into either true or false. Conditional statements are used to see if two items are the same, different, less than or greater than.

(4 < 6) - True
(6 == 4 + 2) - True
(6 <= 6) - True

Functions

Fuctions are pieces of code that can be used multiple times at different points in the program. Fuctions also take in data through a parameters that impacts the way a function works.

function printNumbers() {

   for (i = 0;  i < 5;   i++)    {
        statement  =   "Printed"  +  i  +  "times";
   return statement;
   }

Associative Arrays

Each JavaScript object is an associative array. Each item in an associative array has a key and a value pair. When we create the grades object we pair together the name of what is being graded and the grade. Anything that has a name for the value can be in an associative array.

var grades = {Test1: "98", Quiz1: "68", Test2: "89"}
var text = "";
var x;
for ( x in grades)    {
     text += grades[x];
