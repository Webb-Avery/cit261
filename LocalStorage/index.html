Local Storage

What is local storage?

Local storage will store data with no expiration data. The data will stay around until you delete it so it will be available the next time you open up the web page.

Storing in Local Storage

We can store simple data, arrays, associative arrays and objects in local storage. We will need to store our arrays and objects in JSON form to access them correctly.

Simple Data

localStorage.helloString = "Hello! You added a string!";

Arrays

localStorage.setItem("array", JSON.stringify["first", "second", "third", "forth"]);

Associative Arrays

var associativeArray = { "first" : "Avery", "last" : "Webb", "age" : "21", }
localStorage.setItem("associative", JSON.stringify(associativeArray));

Objects

var Dog = new Object();
Dog.name = "Coco";
Dog.age = "2";
Dog.color = "black";
Dog.gender = "female";
localStorage.setItem("obj", JSON.stringify(Dog));

Retrieving from Local Storage

We will need to transfer our arrays and objects out of JSON form to access them correctly.

Simple Data

output.innerHTML += localStorage.helloString;

Arrays

var savedArray = JSON.parse(localStorage.getItem("array")); output = savedArray;

Associative Arrays

var savedAssociative = JSON.parse(localStorage.getItem("associative"));
Object.keys(savedAssociative).forEach(function(key) {
output.innerHTML += key +": "+ savedAssociative[key]; }, this);

Objects

var savedDog = JSON.parse(localStorage.getItem("obj"));
