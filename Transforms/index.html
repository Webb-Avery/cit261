Transforms!

Designing CSS3 Transforms

We can transform elements of our page to grow or move and add some more motion to our web pages. This allows us to draw attention to areas and material we want the users to see. This will also make us feel cooler.

Defining CSS3 Transforms

There are many ways that we can transform in CSS3. - matrix: This takes in 6 values that will define a 2D transformaion using a matrix of six values.
- translate: this will move the object, the simple translate will move the object on the X and Y axis
    - traslateX : This will only move the object on the X axis
    - traslateY : This will only move the object on the Y axis
    - traslateZ : This will only move the object on the Z axis
    - traslate3D : This will move the object on the X, Y and Z axis
- scale: This will change the size of the shape.     - scaleX : This will only scale the object on the X axis
    - scaleY : This will only scale the object on the Y axis
    - scaleZ : This will only scale the object on the Z axis
    - scale3D : This will scale the object on the X, Y and Z axis
- Skew: This will change the shape to be skewed in a certain direction. Just like the translate and scale properties there are X, Y, Z and 3D versions of the skew property
- Rotate: This will rotate the object a number of degrees. Just like the translate and scale properties there are X, Y, Z and 3D versions of the rotate property

Triggering CSS3 Transforms

You can use transform to simply make your shape always look a little weird or you can add it to a trasition. We will use a trasition to trigger a trasformation. We can use any of the HTML Event Attributes to trigger an Transition. If we want a trasition to happen when the page is loaded for example we can trigger on the onload event. Probably the most common places to add transitions is on Mouse Events.
Here are some mouse events we could trigger our transition on:
- onclick
- ondblclick
- onmouseover
- onmousewheel
- ondrag
- onmousemove
