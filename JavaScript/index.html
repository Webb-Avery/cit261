JavaScript Events for Mobile Devices

onMouseOver Event

function mouseTest() {
var mouse = document.getElementById('mouseOutput');
var mouseOutput = document.getElementById('mouseOutput2');
mouse.innerHTML = "<button class='button'
id='touchButton'>Try touching me</button>";
mouse.onmouseover = function() {
mouseOutput.innerHTML = "THE MOUSE IS TOUCHING ME!!!"; }
mouse.onmouseout = function() {
mouseOutput.innerHTML = "No mouse here... "; }
}

onLoad Event

We use the onload event when we want something to happen as soon as it is loaded. In this case we will simply change a div element to say loaded when the body is finishd loading.

<body onload="loadTest()">

function loadTest() {
document.getElementById("loadOutput").innerHTML = "LOADED!"; }
