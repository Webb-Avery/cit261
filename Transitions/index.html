TRANSITIONS!

Why use Transitions?

We use transitions when we want to change the CSS gradually instead of instantly. If we simply change the CSS instantly it can be strange for the user, but if we transition then they can keep up with how we are changing the design.

Designing CSS3 Transitions

Changing CSS can help us highlight what we want the user to see. For example if the user hovers over some text we can make it bigger. We can change somethings color. These will help the user know what we think is important.

Defining CSS3 Transitions

Transitions are made our of many differnt properties that we can customize.
1. transition-property: This is the property that we will change with the transition.
2. transition-duration: This is how long we want the transition to take to go from the original property to the new property.
3. transition-timing-function: There are several differnt options for how you want to transition.
    Ease starts slow then goes faster then ends slowly.
    Linear is the same speed the whole time.
    Ease-in starts slow but then speeds up.
    Ease-out slows down as the transition progresses
    Ease-in-out starts slow and then speeds up only to slow down again at the end.
    cubic-bezier(n,n,n,n) this allows you to defind the values in a cubic-bezier function.
4. transition-delay: This is how long you want from the time it is triggered to when it starts.

There is also an easy shorthand that allows you to define these four properties in one line.
tranition: This is followed by the property, duration, timeing function and delay seperated by a single space.

div {
transition-property: width;
transition-duration: 5s;
transition-timing-function: ease;
transition-delay: 1s;
}

div {
transition-property: width 5s ease 1s;
}

Triggering CSS3 Transitions

We can use any of the HTML Event Attributes to trigger an Transition. If we want a trasition to happen when the page is loaded for example we can trigger on the onload event. Probably the most common places to add transitions is on Mouse Events.
Here are some mouse events we could trigger our transition on:
- onclick
- ondblclick
- onmouseover
- onmousewheel
- ondrag
- onmousemove
