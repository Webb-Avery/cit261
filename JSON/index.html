JSON

What is JSON?

JSON (JavaScript Object Notation) is a way to transfer data between a browser and a server. The data is formated as a pair of names and values seperated by a colon. Here is an example of what JSON looks like from the server.

var jsonText = '{ "engineName":"Thomas", "trainNumber":7, "cars"=:5, "trainType":"amtrak"}'

JSON Parse

We can use an AJAX request in order to get a JSON from the server. After we do that we need to make the JSON information more easily available to us. We can do this eaily with the JSON.parse() function.

We call JSON.parse() like this:

var trainObj = JSON.parse(jsonText);

After you run that function then you can access each member from the JSON through the trainObj properties.

JSON Stringify

Now that we know how to make an JSON string into an object we are going to learn the opposite. We can take an object and use the JSON.stringify() function to make it into JSON string.

var jsonString = JSON.stringify(trainObj);

Now you can transfer between JSON and objects easily. This will help when you are using JSON to communicate with a server!
