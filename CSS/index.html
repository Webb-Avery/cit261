Each button calls a JavaScript function. Here are a few:

These ones change the syle of the text element that has an id of "toEdit"
function makeBlue() { document.getElementById("toEdit").style.color = "blue"; }
function backgroundGreen() { document.body.style.background = "green"; }
function centerText() { document.getElementById("toEdit").style.textAlign = "center"; }

This one is a bit more complicated. We first have to get the value the user inputted and then we can set the style to that value.
