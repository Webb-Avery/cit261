HTML5 Tags

Video

Let's start with just the video tag.

HTML CODE:
<video>
<source src="disney.mp4" type="video/mp4">
</video>

Here we see some problems.

The default size is pretty big so let's make it smaller using the width and height attributes

HTML CODE:
<video width="300" height = "200">
<source src="disney.mp4" type="video/mp4">
</video>

Now we have another BIGGER problem. The video doen't play. There are a few options for how to make the video play

1. Autoplay the video

HTML CODE:
<video width="300" height = "200" autoplay>
<source src="disney.mp4" type="video/mp4">
</video>

We run into another problem with autoplay. After the auto play the user can't watch the video again. We can fix this by adding controls, as well as autoplay

HTML CODE:
<video width="300" height = "200" autoplay controls>
<source src="disney.mp4" type="video/mp4">
</video>

2. Give the user controls to play the video

HTML CODE:
<video width="300" height = "200" controls>
<source src="disney.mp4" type="video/mp4">
</video>

3. Loop the video

HTML CODE:
<video width="300" height = "200" loop>
<source src="disney.mp4" type="video/mp4">
</video>

I didn't add this video because I didn't want to listen to it on loop always.

Now a black screen looks a bit abnormal when the video is not being played. We fix this by adding a poster attribute that shows an image while the video is not being played.

HTML CODE:
<video width="300" height = "200" controls poster='disney.jpeg'>
<source src="disney.mp4" type="video/mp4">
</video>

Audio

We will start with just the audio tag again.

HTML CODE:
<audio>
<source src="musicBox.mp3" type="audio/mp3">
</audio>

The audio doesn't appear unless we add the controls attribute.

HTML CODE:
<audio controls>
<source src="musicBox.mp3" type="audio/mp3">
</audio>

We can also autoplay and loop the audio. This is done in the same manner as the video tag.

Canvas

The canvas tag is a container for graphics. We must use JavaScript to actually draw the graphics

We can start by simply adding a border to the canvas so we can see it.

<canvas id="exCanvas" width="200" height="100" style="border:1px solid #000000;"> </canvas>

Just like with the video tag we can use the width and height attributes to change the size of our canvas

Now I am going to show you a few things you can do with a canvas, but you can do anything you can dream up with a canvas.

1. Let's start with a simple line.

<script>
var c = document.getElementById("ex1Canvas");
var ctx = c.getContext("2d");
ctx.moveTo(0,35);
ctx.lineTo(200,100)
ctx.stroke();
</script>

2.

<script>
var c = document.getElementById("ex2Canvas");
var ctx = c.getContext("2d");
ctx.fillStyle = "#FF0879";
ctx.fillRect(0,0, 200, 100);
</script>

3.

<script>
var c = document.getElementById("ex3Canvas");
var ctx = c.getContext("2d");
ctx.font = "30px Georgia";
ctx.fillText("Good Job",20,50);
</script>

With JavaScript there is so many things that are posible!
