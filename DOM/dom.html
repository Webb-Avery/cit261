DOM Manipulation

DOM(Document Object Model) makes every element of the HTML into an object. This model allows you to use JavaScript to change HTML and CSS aspects of the web page.

Every time you use the document.getElementByI(id) method you are using the DOM to find the element.

In the next topic "Manipulating CSS Class Properties Using JavaScript" you will learn how to use JavaScript to change aspects of the CSS impacting an object.

Let's start with creating a new element. This code will appear inside a function.

1. Create a new Header 1 tag.

var newHeader = document.createElement("h1");

2. Create a text node that will add text to your h1 tag.

var newText = document.createTextNode("Let's add some text!");

3. Next we will attach the text to the created tag

newHeader.appendChild(newText);

4. Now we will add the element to the document so you can see it on the screen.

document.getElementById("location1").appendChild(newHeader);
