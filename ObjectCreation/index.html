Object Creation

In JavaScript we can create and use objects to store values and methods in an easy to use and manipulate format. In the last topic we talked about what fuctions and variables are and here we will use these principles in objects.

In JavaScript objects have no special keyword they are declared the same way we create any variable with the "var" keyword. Each object can hold as many properties as we can create. Each propertie is a variable holding information. We can also add methods to our object that allow us to do things with the information from the object. Methods are simply functions defined in the object.

Here is an example of a train object:

var train = {
   engineName:"Thomas",
   trainNumber:1,
   cars:5,
   getEngineName() {
       alert(engineName);       
   }
};

The object name is train. The properties are the engineName, trainNumber, and cars variables. This object also contains a function called getEngineName that sends the engine name as an alert to the page.

Instances of an Object

In the previous example we declared all parts of the object at the same time. Another way we can do this by using the "new" keyword. After we do this we can use the dot operator to declare member properties.

Let's try declaring a train object this way:

var train = new Object();
train.engineName = "Edward"; train.trainNumber = 2; train.cars = 8;

We could also create a constructor that would let you populate a function easily, this is a good thing to do if you are created a lot of instances of the object.

function newTrain(name, number, cars) {
    this.engineName = name
    this.trainNumber = number
    this.cars = cars
};

Then we can class the constructor whenever we want to populate and create a new object.

var trainThree = new newTrain("Henry", 3, 4);
